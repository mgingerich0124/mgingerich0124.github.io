What is Hub?
- A hub, also called a network hub, is a connection point for devices in a network. Hubs are devices commonly used to connect segments of a LAN.
Describe Repository and its role?
- A repository is a central place in which some data is kept and maintained in an organized way, usually in computer storage. Repositorys can be a place where multiple databases or files are located for distribution over a network, or a repository can be a location that is directly accessible to the user without having to travel across a network.
